# ConectaDev Project

## 🚀 About the Project
The ConectaDev Project is a complete platform to connect developers, share knowledge, publish posts, and view PDF documents. The project uses a modern architecture with a backend in NestJS, a database managed via Prisma, and a frontend in React + Vite + TailwindCSS.

---

## 📂 Estrutura do Projeto

```
conecta-dev-project/
  ├── src/           # Backend NestJS
  ├── prisma/        # Prisma migrations and schema
  ├── generated/     # Code generated by Prisma
  ├── libs/          # Shared libraries
  ├── front-end/     # React + Vite frontend
  └── ...            # Other files and configs
```

---

## 🛠️ Technologies Used

- **Backend:** [NestJS](https://nestjs.com/), [Prisma ORM](https://www.prisma.io/)
- **Frontend:** [React](https://react.dev/), [Vite](https://vitejs.dev/), [TailwindCSS](https://tailwindcss.com/)
- **Database:** PostgreSQL
- **Others:** Cloudinary (image upload), ESLint, TypeScript

## ⚙️ How to Run the Project

### 1. Clone the repository
```bash
git clone https://github.com/MaironLima/Hosted-Images.git
cd back-end
```

### 2. Configure environment variables
Copy the `.env.example` file to `.env` and fill it with your credentials.

### 3. Install dependencies
```bash
# Backend
cd back-end && npm install
# Frontend
cd front-end && npm install
```

### 4. Run database migrations
```bash
npx prisma migrate dev
```

### 5. Start the backend
```bash
cd front-end 
npm run dev
```

### 6. Start the frontend
```bash
cd front-end
npm run dev
```
